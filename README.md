WX-snake
======

简介
---------
        
实现了贪吃蛇的基本功能，可以拿来继续玩法的开发。对于`初学者`是很好的熟悉`微信小程序`开发流程项目。<br/>

           
            
![github](https://github.com/pagnkelly/wx-snake/blob/master/snake.png)

![github](https://github.com/Savior2016/kui_snake/blob/master/kui_snake.png)
<br/>

在原作者的基础上进行了二次开发：

- 目标是蛇变短;
- 吃紫点变长，绿点变短；
- 蛇身会自己变长，
- 按住加速能延缓变长；
- 按方向键开始游戏
- 由于适配问题，边缘可能有不能触碰的区域
